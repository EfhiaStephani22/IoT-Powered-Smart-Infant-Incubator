## Project Description
This project is an IoT-enabled smart infant incubator designed to monitor and maintain critical health and environmental parameters for premature babies. It provides real-time monitoring and remote alerts to ensure infant safety.

## Parameters Monitored
- Temperature
- Humidity
- Skin temperature
- Heart rate
- Infant weight
- Movement / vibration

## Hardware Components
- Arduino Nano
- MAX30100 (Heart Rate & SpO2 Sensor)
- DHT11 Temperature & Humidity Sensor
- Load Cell with HX711
- Vibration Sensor
- Relay Module
- LCD Display

## Software & Tools
- Embedded C
- Arduino IDE
- Blynk IoT Platform

## IoT Features
- Real-time data monitoring via Blynk mobile app
- Remote alerts for abnormal conditions
- Continuous health tracking

## Applications
- Neonatal Intensive Care Units (NICU)
- Smart healthcare monitoring
- Remote infant care systems
